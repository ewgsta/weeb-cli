# Weeb CLI Dokümantasyonu

**Tarayıcı yok, reklam yok, dikkat dağıtıcı şeyler yok. Sadece siz ve eşsiz bir anime izleme deneyimi.**

## Hoş Geldiniz

Weeb CLI, tarayıcısız ve reklamsız anime izleme deneyimi sunan güçlü bir terminal tabanlı anime akış ve indirme uygulamasıdır. Farklı dillerde birden fazla anime kaynağı desteği, entegre takip servisleri ve gelişmiş indirme yönetimi ile Weeb CLI, hepsi bir arada anime arkadaşınızdır.

## Temel Özellikler

### Çoklu Kaynak Desteği
- Türkçe: Animecix, Turkanime, Anizle, Weeb
- İngilizce: HiAnime, AllAnime
- Almanca: AniWorld
- Lehçe: Docchi

### Akıllı Akış
- MPV ile yüksek kaliteli HLS/MP4 oynatma
- Kaldığınız yerden devam etme
- İzleme geçmişi ve istatistikler
- Bölüm ilerleme takibi

### Gelişmiş İndirmeler
- Aria2 ile çoklu bağlantılı hızlı indirmeler
- yt-dlp ile karmaşık akış desteği
- Eşzamanlı indirmelerle kuyruk sistemi
- Kesintiye uğrayan indirmeleri devam ettirme
- Akıllı dosya adlandırma

### Takipçi Entegrasyonu
- AniList, MyAnimeList ve Kitsu desteği
- OAuth kimlik doğrulama
- Otomatik ilerleme senkronizasyonu
- Bekleyen güncellemeler için çevrimdışı kuyruk

### Yerel Kütüphane
- İndirilen animeleri otomatik tarama
- Harici sürücü desteği (USB, HDD)
- Çevrimdışı anime indeksleme
- Akıllı başlık eşleştirme

### Ek Özellikler
- Çoklu dil desteği (TR, EN, DE, PL)
- Discord Rich Presence
- Sistem bildirimleri
- Etkileşimsiz JSON API
- *arr entegrasyonu için Torznab sunucusu

## Hızlı Başlangıç

```bash
# pip ile kurulum
pip install weeb-cli

# Etkileşimli mod
weeb-cli

# API modu
weeb-cli api search "anime adı"
```

## Dokümantasyon Yapısı

- **Başlangıç**: Kurulum ve ilk kurulum
- **Kullanım Kılavuzu**: Detaylı kullanım talimatları
- **API Referansı**: Tam API dokümantasyonu
- **Geliştirme**: Katkıda bulunma ve geliştirme kılavuzu
- **CLI Referansı**: Komut satırı arayüzü dokümantasyonu
