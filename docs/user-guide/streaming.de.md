# Anime streamen

Erfahren Sie, wie Sie Anime direkt in Ihrem Mediaplayer streamen.

## Einen Stream starten

### Aus Suchergebnissen

1. Suchen Sie nach Anime
2. Wählen Sie Anime aus den Ergebnissen
3. Wählen Sie die Option "Ansehen"
4. Wählen Sie Episode
5. Wählen Sie Qualität/Server
6. Video öffnet sich im MPV-Player

### Aus der Watchlist

1. Hauptmenü → Watchlist
2. Wählen Sie Anime
3. Wählen Sie Episode
4. Stream startet

## Player-Steuerung

### MPV-Tastaturkürzel

- Leertaste: Wiedergabe/Pause
- Links/Rechts-Pfeiltaste: 5 Sekunden zurück/vorwärts spulen
- Auf/Ab-Pfeiltaste: 1 Minute zurück/vorwärts spulen
- [ / ]: Wiedergabegeschwindigkeit verringern/erhöhen
- m: Stummschalten/Stummschaltung aufheben
- f: Vollbild umschalten
- q: Player beenden
- s: Screenshot erstellen
- j: Untertitel durchschalten

### Fortschrittsverfolgung

Weeb CLI automatisch:
- Speichert Ihre Position alle 15 Sekunden
- Markiert Episode als gesehen bei 80% Abschluss
- Synchronisiert Fortschritt mit Trackern (falls konfiguriert)

### Weiterschauen

Wenn Sie eine Episode erneut öffnen:
- Setzt automatisch von der letzten Position fort
- Zeigt Fortsetzungsaufforderung, wenn nahe am Ende
- Löscht Position nach Abschluss

## Qualitätsauswahl

### Verfügbare Qualitäten

Abhängig von Anbieter und Server:
- 1080p (Full HD)
- 720p (HD)
- 480p (SD)
- 360p (Niedrig)
- Auto (adaptiv)

### Qualität wählen

1. Nach Auswahl der Episode
2. Aus verfügbaren Qualitäten wählen
3. Höhere Qualität erfordert schnellere Verbindung

### Qualitätsprobleme

Bei Pufferung:
1. Niedrigere Qualität wählen
2. Internetgeschwindigkeit prüfen
3. Anderen Server versuchen

## Serverauswahl

### Mehrere Server

Die meisten Anbieter bieten mehrere Server:
- Primärer Server (normalerweise am schnellsten)
- Backup-Server
- Verschiedene Hosting-Dienste

### Server wechseln

Wenn Stream fehlschlägt:
1. Zurück zur Episodenauswahl
2. Anderen Server versuchen
3. Verschiedene Server können verschiedene Qualitäten haben

## Untertitel

### Untertitelunterstützung

- Eingebettete Untertitel (falls verfügbar)
- Externe Untertiteldateien
- Mehrsprachige Unterstützung

### Untertitelsteuerung

In MPV:
- j: Durch Untertitelspuren wechseln
- v: Untertitelsichtbarkeit umschalten
- z/x: Untertitelverzögerung anpassen

## Streaming-Probleme

### Video wird nicht abgespielt

1. MPV-Installation überprüfen
2. Andere Qualität/Server versuchen
3. Internetverbindung überprüfen
4. Stream-URL auf Gültigkeit prüfen

### Pufferung

1. Qualitätseinstellung senken
2. Pausieren und puffern lassen
3. Netzwerkgeschwindigkeit prüfen
4. Anderen Server versuchen

### Audio/Video-Synchronisation

1. z/x-Tasten zum Anpassen verwenden
2. Anderen Server versuchen
3. Problem melden, wenn anhaltend

### Keine Untertitel

1. Prüfen, ob Anbieter Untertitel anbietet
2. Anderen Server versuchen
3. j-Taste verwenden, um Untertitelspuren zu durchlaufen

## Erweiterte Funktionen

### Discord Rich Presence

Falls aktiviert, zeigt auf Discord:
- Aktuell angesehenes Anime
- Episodennummer
- Verstrichene Zeit

Aktivieren unter: Einstellungen → Integrationen → Discord RPC

### Wiedergabestatistiken

Ihre Statistiken anzeigen:
- Gesamte Wiedergabezeit
- Gesehene Episoden
- Lieblings-Anime
- Wiedergabeverlauf

Zugriff: Hauptmenü → Watchlist → Statistiken

## Tipps

1. Verwenden Sie Vollbild (f-Taste) für beste Erfahrung
2. Passen Sie Wiedergabegeschwindigkeit mit [ ]-Tasten an
3. Erstellen Sie Screenshots mit s-Taste
4. Verwenden Sie Pfeiltasten für schnelles Spulen
5. Aktivieren Sie Discord RPC, um mit Freunden zu teilen
