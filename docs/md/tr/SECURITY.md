# Güvenlik Politikası

## Desteklenen Sürümler

| Sürüm   | Destek    |

## Bir Güvenlik Açığı Bildirimi

Lütfen güvenlik açıklarını herkese açık GitHub sorunları (issues) aracılığıyla bildirmeyin. Bunun yerine, doğrudan bakımcıya e-posta veya doğrudan mesaj gönderin. Güvenlik açıkları ile ilgili bildirimleri olabildiğince hızlı inceleyip yanıt vereceğiz.
