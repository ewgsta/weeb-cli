# Header UI Component

Uygulama başlığı görüntüleme.

## Genel Bakış

Header modülü şunları sağlar:
- Uygulama markalaşması
- Sürüm görüntüleme
- Görsel ayırıcı

## Uygulama

`weeb_cli/ui/header.py` içinde bulunur.

Biçimlendirilmiş çıktı için Rich kullanır.

## Özellikler

- ASCII art logosu
- Sürüm bilgisi
- Renk biçimlendirme
- Duyarlı düzen
