# Prompt UI Component

Benutzereingabe-Aufforderungen.

## Übersicht

Das Prompt-Modul bietet:
- Texteingabe
- Auswahl-Aufforderungen
- Bestätigungsdialoge
- Mehrfachauswahl

## Implementierung

Befindet sich in `weeb_cli/ui/prompt.py`.

Umschließt Questionary für konsistente Schnittstelle.

## Funktionen

- Eingabevalidierung
- Standardwerte
- Tastaturkürzel
- Fehlerbehandlung
