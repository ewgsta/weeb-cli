# Prompt UI Component

Prompty wejściowe użytkownika.

## Przegląd

Moduł Prompt zapewnia:
- Wprowadzanie tekstu
- Prompty wyboru
- Dialogi potwierdzenia
- Wielokrotny wybór

## Implementacja

Znajduje się w `weeb_cli/ui/prompt.py`.

Opakowuje Questionary dla spójnego interfejsu.

## Funkcje

- Walidacja wejścia
- Wartości domyślne
- Skróty klawiszowe
- Obsługa błędów
