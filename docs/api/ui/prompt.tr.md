# Prompt UI Component

Kullanıcı giriş istemleri.

## Genel Bakış

Prompt modülü şunları sağlar:
- Metin girişi
- Seçim istemleri
- Onay diyalogları
- Çoklu seçim

## Uygulama

`weeb_cli/ui/prompt.py` içinde bulunur.

Tutarlı arayüz için Questionary'yi sarar.

## Özellikler

- Giriş doğrulama
- Varsayılan değerler
- Klavye kısayolları
- Hata işleme
