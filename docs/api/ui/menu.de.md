# Menu UI Component

Interaktives Menüsystem.

## Übersicht

Das Menu-Modul bietet:
- Hauptmenü-Navigation
- Untermenü-Unterstützung
- Tastaturnavigation
- Visuelles Feedback

## Implementierung

Befindet sich in `weeb_cli/ui/menu.py`.

Verwendet Questionary für interaktive Eingabeaufforderungen.

## Funktionen

- Pfeiltastennavigation
- Suchfunktionalität
- Zurück-Navigation
- Exit-Behandlung
