# Menu UI Component

Interactive menu system.

## Overview

The menu module provides:
- Main menu navigation
- Submenu support
- Keyboard navigation
- Visual feedback

## Implementation

Located in `weeb_cli/ui/menu.py`.

Uses Questionary for interactive prompts.

## Features

- Arrow key navigation
- Search functionality
- Back navigation
- Exit handling
