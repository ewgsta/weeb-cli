# Header UI Component

Anwendungs-Header-Anzeige.

## Übersicht

Das Header-Modul bietet:
- Anwendungsbranding
- Versionsanzeige
- Visueller Trenner

## Implementierung

Befindet sich in `weeb_cli/ui/header.py`.

Verwendet Rich für formatierte Ausgabe.

## Funktionen

- ASCII-Art-Logo
- Versionsinformationen
- Farbformatierung
- Responsives Layout
