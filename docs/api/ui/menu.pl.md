# Menu UI Component

Interaktywny system menu.

## Przegląd

Moduł Menu zapewnia:
- Nawigację w menu głównym
- Obsługę podmenu
- Nawigację klawiaturą
- Wizualną informację zwrotną

## Implementacja

Znajduje się w `weeb_cli/ui/menu.py`.

Używa Questionary do interaktywnych promptów.

## Funkcje

- Nawigacja klawiszami strzałek
- Funkcja wyszukiwania
- Nawigacja wstecz
- Obsługa wyjścia
