# Menu UI Component

Etkileşimli menü sistemi.

## Genel Bakış

Menu modülü şunları sağlar:
- Ana menü gezinmesi
- Alt menü desteği
- Klavye gezinmesi
- Görsel geri bildirim

## Uygulama

`weeb_cli/ui/menu.py` içinde bulunur.

Etkileşimli istemler için Questionary kullanır.

## Özellikler

- Ok tuşu gezinmesi
- Arama işlevselliği
- Geri gezinme
- Çıkış işleme
