# Header UI Component

Wyświetlanie nagłówka aplikacji.

## Przegląd

Moduł Header zapewnia:
- Branding aplikacji
- Wyświetlanie wersji
- Wizualny separator

## Implementacja

Znajduje się w `weeb_cli/ui/header.py`.

Używa Rich do sformatowanego wyjścia.

## Funkcje

- Logo ASCII art
- Informacje o wersji
- Formatowanie kolorów
- Responsywny układ
