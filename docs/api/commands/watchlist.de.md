# Watchlist Command

Ansehverlauf und Fortschrittsverfolgung.

## Übersicht

Der Watchlist-Befehl bietet:
- Ansehverlauf
- Fortschrittsverfolgung
- Statistiken
- Schnelle Fortsetzung

## Funktionen

- Letzte Anime anzeigen
- Ansehen fortsetzen
- Abschluss verfolgen
- Statistiken anzeigen

## Implementierung

Befindet sich in `weeb_cli/commands/watchlist.py`.
