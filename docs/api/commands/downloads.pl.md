# Downloads Command

Zarządzanie kolejką pobierania.

## Przegląd

Polecenie Downloads zapewnia:
- Przeglądanie kolejki
- Kontrolę pobierania
- Monitorowanie postępu
- Zarządzanie ponownymi próbami

## Funkcje

- Wyświetl aktywne pobierania
- Wstrzymaj/wznów kolejkę
- Ponów nieudane pobierania
- Wyczyść ukończone elementy

## Implementacja

Znajduje się w `weeb_cli/commands/downloads.py`.
