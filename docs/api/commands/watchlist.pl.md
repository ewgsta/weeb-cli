# Watchlist Command

Historia oglądania i śledzenie postępu.

## Przegląd

Polecenie Watchlist zapewnia:
- Historię oglądania
- Śledzenie postępu
- Statystyki
- Szybkie wznowienie

## Funkcje

- Wyświetl ostatnie anime
- Wznów oglądanie
- Śledź ukończenie
- Wyświetl statystyki

## Implementacja

Znajduje się w `weeb_cli/commands/watchlist.py`.
