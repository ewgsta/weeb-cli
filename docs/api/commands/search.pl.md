# Search Command

Interaktywna funkcja wyszukiwania anime.

## Przegląd

Polecenie Search zapewnia:
- Wyszukiwanie u wielu dostawców
- Interaktywne wyniki
- Historię wyszukiwania
- Szybki dostęp

## Implementacja

Znajduje się w `weeb_cli/commands/search/`:
- `search_handlers.py`: Logika wyszukiwania
- `anime_details.py`: Wyświetlanie szczegółów
- `episode_utils.py`: Wybór odcinka
- `stream_utils.py`: Wybór strumienia
- `watch_flow.py`: Przepływ oglądania
- `download_flow.py`: Przepływ pobierania

## Funkcje

- Wybór dostawcy
- Filtrowanie wyników
- Śledzenie historii
- Szybka nawigacja
