# Search Command

Interaktive Anime-Suchfunktionalität.

## Übersicht

Der Search-Befehl bietet:
- Multi-Provider-Suche
- Interaktive Ergebnisse
- Suchverlauf
- Schnellzugriff

## Implementierung

Befindet sich in `weeb_cli/commands/search/`:
- `search_handlers.py`: Suchlogik
- `anime_details.py`: Detailanzeige
- `episode_utils.py`: Episodenauswahl
- `stream_utils.py`: Stream-Auswahl
- `watch_flow.py`: Ansehen-Workflow
- `download_flow.py`: Download-Workflow

## Funktionen

- Provider-Auswahl
- Ergebnisfilterung
- Verlaufsverfolgung
- Schnellnavigation
