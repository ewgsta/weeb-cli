# Downloads Command

İndirme kuyruğu yönetimi.

## Genel Bakış

Downloads komutu şunları sağlar:
- Kuyruk görüntüleme
- İndirme kontrolü
- İlerleme izleme
- Yeniden deneme yönetimi

## Özellikler

- Aktif indirmeleri görüntüle
- Kuyruğu duraklat/devam ettir
- Başarısız indirmeleri yeniden dene
- Tamamlanan öğeleri temizle

## Uygulama

`weeb_cli/commands/downloads.py` içinde bulunur.
