# Downloads Command

Download queue management.

## Overview

The downloads command provides:
- Queue viewing
- Download control
- Progress monitoring
- Retry management

## Features

- View active downloads
- Pause/resume queue
- Retry failed downloads
- Clear completed items

## Implementation

Located in `weeb_cli/commands/downloads.py`.
