# Watchlist Command

İzleme geçmişi ve ilerleme takibi.

## Genel Bakış

Watchlist komutu şunları sağlar:
- İzleme geçmişi
- İlerleme takibi
- İstatistikler
- Hızlı devam etme

## Özellikler

- Son anime'leri görüntüle
- İzlemeye devam et
- Tamamlanmayı takip et
- İstatistikleri görüntüle

## Uygulama

`weeb_cli/commands/watchlist.py` içinde bulunur.
