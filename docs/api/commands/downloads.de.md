# Downloads Command

Download-Warteschlangenverwaltung.

## Übersicht

Der Downloads-Befehl bietet:
- Warteschlangenanzeige
- Download-Steuerung
- Fortschrittsüberwachung
- Wiederholungsverwaltung

## Funktionen

- Aktive Downloads anzeigen
- Warteschlange pausieren/fortsetzen
- Fehlgeschlagene Downloads wiederholen
- Abgeschlossene Elemente löschen

## Implementierung

Befindet sich in `weeb_cli/commands/downloads.py`.
